■概要
ドッペルゲンガーを探すアプリ
世界には同じ顔の人間が3人いるといわれている。
Google Cloud Machine Learning
Microsoft Face API

↑のようなAPIで似ている顔を探してみるようなアプリ。


■プライバシーについて
データの構造は顔写真と任意のＩＤのみ。逆にそれ以外の情報は一切保持しない。
　→世界中の大企業が顔と名前、住所、電話番号、カード番号といったデータを保持している。
　→今更顔だけの情報がネットに出回ることの何が問題なのだろうか？
　→写真の背景は白、瞳に移る景色も特定できないようなものにすれば良い。そのあたりの写真アップロード時の制約を強めにすればよい。

■課題
データレコード最大数は単純に考えると世界の人口と同じ。
それ以上になることはない想定
